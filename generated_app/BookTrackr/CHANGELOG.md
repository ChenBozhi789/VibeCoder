
# Changelog

## v0.1.0 - 2024-07-25

### Added
- Initial implementation of the BookTrackr application.
- Core features:
    - Create, read, update, and delete books.
    - Track reading progress.
    - Mark books as finished.
    - Search and filter books.
    - Import and export data as JSON.
    - Clear all data.
- State management using a global `state` object.
- Data persistence using `localStorage`.
- Single-page application architecture with three views: Home, Form, and Settings.
