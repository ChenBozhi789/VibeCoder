# CHANGELOG

All notable changes to GreenTodo.

## 0.1.0 - Initial Functional Implementation
- Replaced ui/index.html with SPA structure (Home, Tasks, Search).
- Implemented full task management (CRUD) with localStorage persistence.
- Added filters: status, priority, due (overdue/today/week).
- Added search (title/description) and sorting (due date, priority, created).
- Implemented bulk clear of completed tasks.
- Implemented data portability: export (download JSON) and import (from JSON file).
- Added inline form validation and error messages.
- Improved accessibility: ARIA labels, skip link, modal ESC handling.
- Updated ui/css/style.css with responsive, accessible styles.
- Updated ui/js/main.js with state management, rendering, and event wiring.
