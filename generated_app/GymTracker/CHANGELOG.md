
# Changelog

## v0.1.0 - 2023-10-27

### Added
- Initial implementation of the GymTracker application.
- Created the core HTML structure for the list, detail, and settings views.
- Implemented the core JavaScript logic in `js/main.js` including:
    - State management for logs, current view, and editing state.
    - Data persistence using `localStorage`.
    - Navigation between views.
    - CRUD (Create, Read, Update, Delete) functionality for exercise logs.
    - Dynamic form for adding and removing sets.
    - Search and filter functionality by exercise name and date.
    - Import and export of data as a JSON file.
    - "Clear All Data" functionality.
- Added basic CSS styling for a clean and functional user interface.
