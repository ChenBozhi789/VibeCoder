
# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2024-05-20

### Added
- Initial implementation of the WantList application.
- Feature: Item management (Create, Read, Update, Delete).
- Feature: Search by item title.
- Feature: Filter items by price.
- Feature: Data persistence using `localStorage`.
- Feature: Import and export data as JSON.
- Feature: Clear all data.
- Basic styling for the application.
