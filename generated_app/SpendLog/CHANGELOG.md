
# Changelog

## v1.0.0 - (2023-10-27)

### Added
- Initial implementation of the SpendLog application.
- Feature: Add a new expense with name, amount, and category.
- Feature: View a list of all expenses.
- Feature: Delete an expense from the list.
- Feature: Display total expenses and a breakdown by category.
- Data persistence using `localStorage`.
- Basic input validation for the expense form.
