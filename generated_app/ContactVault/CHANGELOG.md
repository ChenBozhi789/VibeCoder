
# Changelog

## v0.1.0 - 2024-05-22

### Added
- Initial implementation of the ContactVault application.
- Core features:
    - Contact Management (Create, Read, Update, Delete)
    - List & Preview
    - Search & Filter
    - Data Portability (Import/Export JSON)
    - Basic Validation
- Implemented data persistence using `localStorage`.
- Created a three-view UI (List, Detail, Settings).
